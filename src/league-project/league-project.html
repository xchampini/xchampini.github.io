League of Legends Matches Calendar View
* Typescript | Express | RiotAPI
* Still Ongoing!
There are a lot of applications online that display league of legends match data, however I noticed these applications always displayed the most recent matches in a basic list format. I wanted to create a more visual representation of how a user has performed each day (as well as get more specific info that isn't available in the other applications) so I created a calendar view that accomplishes this. That, and I wanted some experience building a full stack application.
When opening the application, the user is first prompted to enter their player's name and region. When the user submits this form, the application will fetch the user's match history and display it in a calendar view. Each day on the calendar will show the number of matches played, as well as the user's performance in those matches (e.g., wins/losses, KDA, etc.). It also displays the user's winrate for a certain day.
Work on this application is still on going - there are still features I want to implement, as well as improve the calendar view (e.g. split it into a monthly view, add a feature to get a summary of the user's performance for a specific match, display the user's most strongest champions, among other features)
